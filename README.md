Yii 2 Workshop RBAC DB
===============================

### Install
- create db yii2-rbac-db
- import `yii2-rbac-db.sql` to db
- extract zip file and cd  to project rbac-db
- config db in common/config/main-local.php
- run php init
```
php /path/to/yii-application/init
```
select  development

install packet
```
composer update
```
